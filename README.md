# 🚗 ZEN AUTO - Excellence Automobile

**ZEN AUTO** est une plateforme web moderne dédiée à la vente de véhicules de prestige et de luxe à Yaoundé, Cameroun. Ce projet a été réalisé dans le cadre de l'examen de Bachelor 1 en Informatique chez **Keyce**.

## 👥 Notre Équipe (Groupe de 7 membres)
En tant que chef de projet, j'ai coordonné une équipe de 7 personnes pour mener à bien ce développement :

- **Hamidou Boubakari (Chef de projet)** : Structure globale, page Accueil (`index.html`), Boutique (`boutique.html`) et gestion du dépôt GitHub.
- **Mekoa** : UI/UX Design, identité visuelle (`style.css`) et interactivité (`script.js`).
- **Le Roi** : Rédaction des contenus, pages institutionnelles (`a-propos.html`) et relation client (`contact.html`).
- *(Note : Les autres membres ont participé aux phases de tests, de recherche documentaire et de saisie des données techniques des véhicules).*

## ✨ Fonctionnalités du Site
- **Catalogue de Prestige** : Exposition de 10 véhicules de luxe avec fiches techniques et prix en FCFA.
- **Design Zen & Moderne** : Interface épurée avec un thème sombre et bleu azur pour une expérience utilisateur premium.
- **Responsive Design** : Site entièrement consultable sur ordinateur, tablette et smartphone.
- **Système de Contact** : Formulaire intégré pour les demandes de devis et rappels clients.

## 🛠️ Technologies Utilisées
- **HTML5 / CSS3** : Pour une structure sémantique et un design soigné.
- **JavaScript** : Pour les animations et la validation des formulaires.
- **Git & GitHub** : Pour le travail collaboratif et le versioning.
- **GitHub Desktop** : Outil principal de synchronisation de l'équipe.

## 🚀 Gestion du Projet & Leadership
La réalisation de ce projet a nécessité une coordination rigoureuse, notamment pour surmonter les défis techniques liés à la synchronisation :

1. **Formation Technique** : J'ai organisé des sessions de **partage d'écran via WhatsApp** pour former les membres de l'équipe à l'utilisation de GitHub (Cycle : Fetch -> Pull -> Commit -> Push).
2. **Résolution de Conflits** : Gestion des conflits de versions et des problèmes de connectivité internet pour assurer la continuité du travail.
3. **Contrôle Qualité** : Revue du code de chaque membre pour garantir une uniformité visuelle et fonctionnelle sur l'ensemble du site.
